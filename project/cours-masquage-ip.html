Le masquage d'adresse IPIP address masking

Un masque sépare une adresse IP en deux parties : réseau et hôte. Manipulez le simulateur ci-dessous pour voir, bit par bit, où passe la frontière.A mask splits an IP address into two parts: network and host. Use the simulator below to see, bit by bit, where the boundary lies.

1. Réseau ou hôte : la frontière1. Network or host: the boundary

Une adresse IPv4 fait 32 bits. Le préfixe CIDR (le /24 de 192.168.1.0/24) indique combien de bits, en partant de la gauche, désignent le réseau. Les bits restants désignent l'hôte à l'intérieur de ce réseau.An IPv4 address is 32 bits. The CIDR prefix (the /24 in 192.168.1.0/24) says how many bits, from the left, denote the network. The remaining bits denote the host within that network.

i

Le masqueThe mask

Le masque est l'adresse où les bits réseau valent 1 et les bits hôte valent 0. /24 ⇒ 255.255.255.0.The mask is the address where network bits are 1 and host bits are 0. /24 ⇒ 255.255.255.0.

2. Ce que le masque détermine2. What the mask determines

  • Adresse réseau : bits hôte tous à 0 — l'identité du réseau.Network address: host bits all 0 — the network's identity.
  • Diffusion : bits hôte tous à 1 — atteint tous les hôtes.Broadcast: host bits all 1 — reaches every host.
  • Hôtes utilisables : \(2^{(32-p)} - 2\) (on retire réseau et diffusion).Usable hosts: \(2^{(32-p)} - 2\) (network and broadcast excluded).
!

Cas limitesEdge cases

/31 n'a pas de diffusion (2 hôtes, liaison point-à-point, RFC 3021) et /32 désigne un seul hôte./31 has no broadcast (2 hosts, point-to-point link, RFC 3021) and /32 denotes a single host.

Préfixe = frontièrePrefix = boundary

Le /p fixe le nombre de bits réseau. Plus p est grand, plus le réseau est petit (moins d'hôtes).The /p sets the number of network bits. The larger p, the smaller the network (fewer hosts).

Réseau & diffusionNetwork & broadcast

Hôtes à 0 ⇒ adresse réseau ; hôtes à 1 ⇒ diffusion. Ni l'une ni l'autre n'est attribuable à une machine.Host bits 0 ⇒ network address; host bits 1 ⇒ broadcast. Neither is assignable to a machine.

Compter les hôtesCounting hosts

\(2^{(32-p)} - 2\). Exemple : /26 ⇒ \(2^{6}-2 = 62\) hôtes.Example: /26 ⇒ \(2^{6}-2 = 62\) hosts.
